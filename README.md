# aor-graph-cool-client
 
based on 
https://github.com/marmelab/aor-simple-graphql-client

for 
https://github.com/marmelab/admin-on-rest

## Installation

Install with:

```sh
npm install --save aor-graph-cool-client
```

or

```sh
yarn add aor-graph-cool-client 
```
